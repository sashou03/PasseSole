# PasseSole
Projet de TCIEL

BUT DU JEU
Chaque joueur reçoit une mission secrète qu’il doit accomplir dans la vie réelle (dans une salle, une soirée, un cours, etc.) sans se faire remarquer.
Les missions peuvent être par exemple : dire une phrase particulière, faire une action discrète, provoquer une réaction chez quelqu’un, etc.

Chaque mission doit être réalisée dans un délai de 10 minutes maximum après son attribution.

Une fois que le joueur pense avoir réussi sa mission, les autres joueurs doivent voter pour confirmer ou refuser la réussite.

Si la majorité vote "réussi", la mission est validée et le joueur gagne un point.

Le créateur de la partie possède des droits spéciaux : il peut arrêter ou écourter le temps d’une mission si nécessaire.
